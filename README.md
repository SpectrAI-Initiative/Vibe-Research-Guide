## What is Vibe Research

用 LLM Agent 自动化科研流程：文献调研 → idea 发现 → 实验执行 → 论文撰写 → 评审反馈。

本仓库服务两类读者：**想上手的人**（最快学会使用）和**做研究的人**（跟踪最新进展）。

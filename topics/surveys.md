<h1 align="center">Vibe Research Guide<br>综述篇 Surveys</h1>

> 综述类论文帮助你快速建立对 **Vibe Research** 全景的认知：从 LLM 在科学发现中的角色演化，到 AI Scientist 的完整科研生命周期，再到系统设计、基准与伦理。建议入门读者**按顺序通读**这三篇，用 1-2 天建立领域心智模型。

<section id="survey-papers"></section>

## 论文列表

| # | Paper | URL | Type | Why read / 为什么读 | Stage | Keywords |
|---|---|---|---|---|---|---|
| 1 | From Automation to Autonomy: A Survey on Large Language Models in Scientific Discovery | [arXiv](https://arxiv.org/abs/2505.13259) | Survey | 建立全局框架，理解 LLM in Scientific Discovery 从 Tool → Analyst → Scientist 的演化路径。入门第一篇。 | Beginner | LLM4Science, evolution |
| 2 | LLM Agents as AI Scientists: A Survey | [OpenReview](https://openreview.net/pdf?id=bfdUWy6rUA) | Survey | 从科研全流程理解 agent：hypothesis discovery → experiment → paper writing → peer review，覆盖端到端生命周期。 | Beginner | AI Scientist, lifecycle |
| 3 | A Survey of LLM-based Scientific Agents | [arXiv](https://arxiv.org/abs/2503.24047) | Survey | 更偏系统视角：benchmark、architecture、applications、ethics。适合从入门过渡到进阶。 | Beginner-Intermediate | system design, benchmark |
| 4 | The Landscape of Emerging AI Agent Architectures for Reasoning, Planning, and Tool Calling | [arXiv](https://arxiv.org/abs/2404.11584) | Survey | Agent 架构全景综述：涵盖单 Agent、多 Agent、工具调用等架构模式。帮助你理解 research agent 背后的通用设计范式。 | Beginner-Intermediate | agent architecture, reasoning, planning |
| 5 | A Survey on Large Language Model based Autonomous Agents | [arXiv](https://arxiv.org/abs/2308.11432) | Survey | 经典 LLM Agent 综述，系统梳理了 Agent 的构建（profile、memory、planning、action）与评估。虽非专注科研场景，但提供了理解 research agent 必需的基础框架。 | Beginner | autonomous agents, agent construction, evaluation |

<section id="survey-reading-tips"></section>

## 阅读建议

1. **论文 #1** 适合作为你的第一篇阅读——它提供了从"工具"到"科学家"的清晰演化框架，读完后你会对整个领域有一张地图。
2. **论文 #2** 补充了端到端流程细节：从假设发现到论文写作和同行评审，非常适合理解"完整科研 Agent 应该做什么"。
3. **论文 #3** 更适合你准备开始动手做项目时回来细读——它的系统设计和 benchmark 讨论有很高的工程参考价值。
4. **论文 #4** 适合关注 Agent 架构设计的读者——它不限于科研场景，但架构模式可以直接迁移到 research agent 设计中。
5. **论文 #5** 虽然发表较早（2023），但作为 LLM Agent 方向的经典综述，至今仍是理解 Agent 组成要素的最佳参考。
